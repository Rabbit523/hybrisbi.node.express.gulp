About This Page

This page is for changing the targets on the generated PDF score card
  • Select target dropdown to select target to change
  • Green threshold: value below which the target should be green
  • Yellow threshold: value below which the target should be yellow
  • Update target button to save the changes
Note: values above yellow threshold will be red
Success! Target information updated.
Failure! Check the information you entered.
